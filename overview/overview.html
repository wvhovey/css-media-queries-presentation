What exactly is a media query?

A media query is code that uses the @media rule to include CSS properties if its condition evaluates to 'true'.

A media query contains 4 parts: the required @media rule, a media type, media features, and the css properties or code.

Syntax

@media not|only mediatype and (mediafeature and|or|not mediafeature) { CSS-Code; }
