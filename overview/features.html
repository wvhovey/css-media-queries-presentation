Media Features

There are TONS of media features that can be used in media queries. These features are the expression that must evaluate to true for a change to take place. These features can be viewport height, width, orientation, color indexes and more!

Syntax

(mediafeature and|or|not mediafeature)

Example

(max-width: 600px and min-width: 1000px)
