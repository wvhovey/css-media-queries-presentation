Media Types

The part of a media query that follows the rule is the type. As its name suggests, it is a piece of code the allows the coder to specify which type of media we are changing. There are four main keywords that specify these different types: all, print, screen, and speech.

'all'

This keyword is the default setting for media queries and can be used to manipulate all types of media.

'print'

This keyword is used for printers and allows changes to be made to your site if it is being printed.

'screen'

This is used specifically for types of screens whether it be a computer, phone, or tablet.

'speech'

This keyword is used for screenreaders that speak aloud what is written on your site.

Syntax

@media not|only mediatype and

Example

@media only screen and
